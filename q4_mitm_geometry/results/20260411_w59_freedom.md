# W[59] Has 7-Bit Freedom in Round-61 Closure (Cert)

## Discovery

Tested single-bit XOR perturbations of cert's W1[57], W1[58], W1[59]
(while keeping the other two at cert values). Results:

| Word perturbed | Bits preserving round-61 closure |
|---|---|
| W1[57] | **0 of 32** |
| W1[58] | **0 of 32** |
| **W1[59]** | **8 of 32** (bits 11, 12, 16, 17, 28, 29, 31) |

## W[59] freedom details

| Bit | Round-61 matches |
|---|---|
| 11 | 8192 |
| 12 | 8192 |
| 16 | 8192 |
| 17 | 4096 (partial!) |
| 28 | 8192 |
| 29 | 8192 |
| 31 | 8192 |

7 bits give the full 8192 matches, 1 bit (17) gives half (4096).
This is a 7-bit DOF in W[59].

## Combinatorial implication

For the cert prefix structure (cert W[57], cert W[58]), there are at
least 2^7 = 128 valid W[59] choices (just from single-bit perturbations).
With multi-bit combinations, the total may be 2^7 to 2^14 valid W[59]
values (depending on whether the perturbations compose).

Combined with the 8192 = 2^13 W[60] family, the cert structure
contains 2^20 to 2^27 valid (W[59], W[60]) pairs that close round-61.

Of these, only a fraction also pass round-62 and round-63. The cert
is one specific (W[57], W[58], W[59], W[60]) that passes all three.

## Why W[59] is special

- W[57]: cascade trigger — fixed by da[57]=0 condition
- W[58]: bridge — fixed by db[58]=0 cascade propagation
- W[59]: enters the schedule rule for W[61] = sigma1(W[59]) + const
  — and ALSO enters the round-59 state via T1_59
- W[60]: free word for de[60]=0 cascade trigger

The round-61 closure constraint is:
  cascade_dw(state60) = sigma1(W2[59]) - sigma1(W1[59]) + sched_const_diff

W[57] and W[58] only enter the LHS (via state60). W[59] enters both
sides — the LHS via state60 and the RHS via sigma1.

The 8-bit freedom in W[59] reflects bit positions where the
sigma1(W2[59]) - sigma1(W1[59]) value AND the corresponding state60
change "cancel out" in the cascade equation.

## Constructive search idea

For each cert-like (W[57], W[58]) prefix:
1. Find the W[59] family (~2^7 to 2^14 valid values)
2. For each, find the W[60] family (~2^13 valid values)
3. Test round-62 closure on all (W[59], W[60]) pairs

If round-62 closure happens with frequency p ~ 2^-32, we expect
2^7 × 2^13 × 2^-32 = 2^-12 collisions per (W[57], W[58]) pair. So
we need ~2^12 = 4096 valid (W[57], W[58]) pairs to find 1 collision.

Need to characterize what makes (W[57], W[58]) "valid" for the cert
structure. That's the next question.

## Evidence level

**STRONG EVIDENCE**: Direct full 2^32 W1[60] enumeration on 64 prefix
variations (32 W[57] perturbations + 32 W[58] perturbations + 32 W[59]
perturbations = 96 total). The 8/32 W[59] result is exact.

The asymmetry (W[57], W[58]: 0/32 vs W[59]: 8/32) is the structural
discovery. This identifies which words have local freedom.
