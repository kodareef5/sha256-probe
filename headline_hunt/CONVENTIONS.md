# Headline Hunt Conventions

## Result And Comms Dates

Local laptop coordination for the 2026-04-30 evening run uses EDT wall time.
Some files from the immediately preceding macbook handoff are named
`20260501_*` even though this Codex session's local wall clock is still
2026-04-30 EDT.

Those filenames are left intact because they are already referenced by result
chains, comms summaries, and fleet pull context. Treat the date in the filename
as the producing session's handoff label, not as a strict local-wall-clock claim.
When adding new files from this session, use the local EDT date unless an
existing chain intentionally uses another date label.
