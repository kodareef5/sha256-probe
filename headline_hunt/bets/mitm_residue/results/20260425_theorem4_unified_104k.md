## Computation

The corpus stores `iv_63 = (a, b, c, d, e, f, g, h)_63`. Shift register propagation gives:
```
a_60 = d_63   (since d_63 = c_62 = b_61 = a_60)
a_61 = c_63
a_62 = b_63
e_63 = iv[4]
```

So `dT2_63 = dSigma0(a_62) + dMaj(a_62, a_61, a_60)` is computable directly from the corpus IVs without any extra forward computation.
