# Low-HW de58 values per candidate
**2026-04-25** — sr61_n32 — bonus probe on de58 image structure.

## Question

Which candidates' de58 images contain **low Hamming-weight** values? Low-HW modular differences propagate with fewer carry positions through subsequent rounds (Lipmaa-Moriai). If "low-HW de58 = structurally easier sr=61 SAT search" is a real principle, then candidates whose image hits low-HW values are advantaged.

## Method

For each candidate: sample 131,072 W57 → de58 image. Find the de58 value(s) with minimum Hamming weight.

## Two competing structural extremes

The de58 analysis now identifies **two different "best candidates"** depending on the structural principle:

1. **Compression-extreme** (image-size predictor): bit-19 with image=2^8, narrow span, away from low-HW.
2. **Low-HW-extreme** (Lipmaa-Moriai predictor): bit13_m4e560940 / bit17_m427c281d / msb_m189b13c7 with HW=3 reachable.

Without further measurement, we don't know which (if either) actually predicts sr=61 SAT-friendliness.

## Caveat — speculative principle

The "low-HW de58 = better SAT search" claim is intuitive but unproven for cascade-DP r=61. To confirm, one would need to:
1. Sample W57 such that de58 lands at a low-HW value (which is rare among W57s for these candidates).
2. Check whether da_61 = 0 is more likely conditional on de58 being low-HW.

This is a future experiment, not concluded here.
