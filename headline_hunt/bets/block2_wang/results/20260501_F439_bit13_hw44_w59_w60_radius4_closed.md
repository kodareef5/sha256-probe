## Tooling Note

The generic Hamming-ball enumerator now accepts `--slots`, so exact local
checks can target selected W words without one-off scripts. Default behavior
remains the full W57..W60 128-bit ball.

Example:

```bash
python3 headline_hunt/bets/block2_wang/encoders/enumerate_hamming_ball.py \
  --candidate bit13_m916a56aa \
  --init-W 0x5228ed8d,0x61a1a29c,0xea6a8c21,0x3dbfd852 \
  --init-hw 44 \
  --max-radius 4 \
```
